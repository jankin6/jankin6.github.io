“你和萤火虫有两个共同点，在我的眼里都会发光，同时，都已经很多年没见了。”

——《夜、萤火虫和你》网易云音乐热评

1. 多级标题

方法一： #加上空格

# 这是第一级标题

## 这是第二级标题

方法二：快捷键：ctrl+ 1~6 清除ctrl+ 0

方法三：选择文字后点击菜单栏段落选择标题级数

2. 列表

1. 有序列表

语法： 数字加.加空格 1.

快捷键：ctrl+shift+[

有序列表中嵌套有序列表 ->选择后按tab键实现降级

按shift+tab实现升级

2. 无序列表

语法：加空格 / -加空格 *

快捷键：ctrl+shift+]

  1. 青菜

  2. 水果

    • 苹果
    • 菠萝

    选择后按tab实现降级

3. 有序与无序的切换

有序快捷键：ctrl+shift+[

无序快捷键：ctrl+shift+]

配合tab和shift+tab

实现无序与有序的交错使用

有序与无序也可以选择后在菜单段落中选择有序列表和无序列表

4. 任务列表

语法：-加空格加[加空格加]加空格 - [ ]

还需在开头，如果前面有文字就无法实现

  • 任务一

  • 任务二

  • 任务三

    回车加回车取消任务列表

    方法二：选择后点击菜单段落选择任务列表

3. 插入表格

语法：|加xx加|…然后回车就行

快捷键：ctrl+t

在表格中点击任意单元通过快捷键ctrl+回车即可添加新的一行

2. 代码块

5. 插入图片

粗暴的直接拖拽

语法创建：![自定义图片名字](地址)

快捷键：ctrl+shift+i

6. 加粗，倾斜，删除线，下划线，高亮，上下标

  • **加粗** 加粗

  • *倾斜* 倾斜

  • 加粗倾斜 加粗倾斜

  • ~~删除线~~ 删除线

  • 下划线快捷键ctrl+u 下划线

  • ==高亮== ==高亮== （不过好像在网页上并未显示出高亮效果）

  • ^上标^ ^上标^

  • ~下标~ 下标

  • --- 分割线 也可***,___

  • 引用1

  • 表情符号

输入:xxx:

一些常用例子:

:smile: :laughing: :dizzy_face: :sob: :sweat_smile: :cry: :triumph: :heart_eyes: :relaxed: :sunglasses: :weary: :+1: :-1: :100: :clap: :bell: :gift: :question: :bomb: :heart: :coffee: :cyclone: :bow: :kiss: :pray: :sweat_drops: :hankey: :exclamation: :anger:

  • 实空心点

  • +号和空格

8. 内外部跳转

快捷键：ctrl+k

1. 外部跳转–超链接

格式：[xxx]()

百度

2. 内部跳转

格式：[xxx](#要去的目的地--标题）

我要去插入表格那里

3. 自动链接

使用 <> 包括的 URL 或邮箱地址会被自动转换为超链接：

https://www.baidu.com

123@email.com

9. 快捷键汇总

Ctrl 0 到 ：普通文本、一级～六级标题；Ctrl+1~6

Ctrl B：加粗

Ctrl I：斜体

Ctrl U：下划线

Shift Alt 5：删除线

Shift+Ctrl+Tab键上面那个键：行内代码块

Ctrl K：超链接

Ctrl T：表格

Ctrl Shift Q：引用

Shift Ctrl I：图片

Shift Ctrl M：公式块
